WEBSITE PORTOFOLIO - IRSYAD NUR HAKIM 2026

Isi paket:
1. index.html
2. style.css
3. script.js
4. assets/images/page-01.jpg sampai page-10.jpg

Cara membuka:
1. Extract ZIP.
2. Buka file index.html di browser.
3. Untuk mengedit teks, buka index.html dengan VS Code.
4. Untuk mengedit warna dan layout, buka style.css.
5. Untuk mengedit interaksi modal dan menu mobile, buka script.js.

Bagian yang sebaiknya kamu edit:
1. Link website irsyadnurhakim.com jika belum aktif.
2. Link Instagram @ir_nurhakim jika ingin diganti.
3. Email jika ingin pakai email bisnis.
4. Deskripsi project jika ingin dibuat lebih detail.
5. Tambahkan project baru di section Selected Works.

Cara upload ke hosting gratis:
- Netlify: drag folder hasil extract ke Netlify Drop.
- Vercel: upload sebagai project static.
- GitHub Pages: upload semua file ke repository, lalu aktifkan Pages.

Catatan:
Website ini dibuat dari portfolio PDF yang kamu upload. Gaya visual dibuat mengikuti identitas PDF: background gelap, aksen merah, pill outline putih, dan aksen hijau.
